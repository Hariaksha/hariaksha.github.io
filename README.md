# hariaksha.github.io

Personal academic website for **Hari Gunda** — Boren & Gilman Research Scholar, Data Science & Foreign Languages student at the University of Alabama, and aspiring PhD researcher in Sustainable Development.

🌐 Live site: [hariaksha.com](https://hariaksha.com)

---

## About

This is the source code for my personal academic website. It is a fully static, single-page site built with vanilla HTML, CSS, and JavaScript — no frameworks, no build tools, no dependencies beyond a few CDN-loaded fonts and icons.

The `redesign` branch contains a full ground-up redesign of the site with a cleaner layout, dark mode support, collapsible sections, and improved mobile responsiveness.

---

## Features

- **Dark / Light mode toggle** with persistent preference
- **Scroll progress bar** and back-to-top button
- **Responsive navigation** with hamburger menu for mobile
- **Collapsible sections** for Experience and Awards (show/hide extra items)
- **Professional / Fun experience toggle** to switch between views
- **Research modal popups** for paper abstracts and award details
- **Language proficiency bars** with ACTFL / CEFR certifications
- **Featured press & podcast cards** in a horizontally scrollable track
- **Meeting scheduler integration** (zcal) for 30- and 60-minute bookings
- **SEO-optimized** meta tags, Open Graph, Twitter Card, sitemap, and robots.txt
- **Custom domain** via `CNAME` (`hariaksha.com`)
